# Sprint 9: Nocions i coneixements bàsics de Python

# Creació de funcions, estructures de dades i bucles

## Descripció

Aquest sprint consisteix en resoldre diversos problemes de la vida quotidiana aplicant estructures de dades i control en Python. El client de l’empresa ha demanat una sèrie de programes senzills que li facilitin processos habituals, i s’espera que cada funció inclogui tant el codi com la seva execució amb output demostratiu.

Els objectius principals són consolidar els fonaments de Python i practicar:

- Creació de funcions reutilitzables

- Ús de diferents estructures de dades (llistes, diccionaris, tuples…)

- Control de flux i validació d’entrada

- Automatització de processos simples


# Nivell 1

## Calculadora de l’Índex de Massa Corporal (IMC)

Calcula l’IMC a partir de dades introduïdes per l’usuari/a.

Classifica el resultat segons les categories establertes.

Inclou validació de dades per assegurar valors raonables.

##  Convertidor de temperatures

Conversió entre almenys dues unitats (Celsius, Fahrenheit, Kelvin, etc.)

Retorna totes les conversions dins d’un objecte guardable (llista, diccionari…).

Valida les dades introduïdes.

## Comptador de paraules

Mostra la freqüència de cada paraula en un text donat.

Gestiona casos especials per evitar errors de càlcul.

Extra: Calcula la llargada mitjana de les paraules del text.

## Diccionari invers

Intercanvia claus i valors d’un diccionari.

Si hi ha duplicats en els valors, mostra un avís i els valors duplicats.


# Nivell 2

## Comptador i endreçador de paraules

Llegeix un arxiu TXT i calcula la freqüència de cada paraula.

Ordena el resultat alfabèticament per la primera lletra de cada paraula.

## Conversió de tipus de dades

A partir d’una llista complexa, separa els elements que es poden convertir a float dels que no.

Gestiona tuples i llistes dins de la llista principal.


# Nivell 3

## Generador de contrasenyes

Genera contrasenyes segures amb paràmetres configurables: longitud, majúscules, minúscules, números i caràcters especials.

Les contrasenyes són realment aleatòries i es poden emmagatzemar.

Extra: Opció de copiar la contrasenya al porta papers automàticament.

## Processament de dades simple

Analitza un fitxer d’històric de partits de futbol (historic_partits.txt).

Retorna:

- Nombre total de gols per equip

- Equip més golejador

- Equip més golejat

- Classificació global (3 pts victòria, 1 pts empat, 0 pts derrota)
