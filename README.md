## Descripció / Descripción / Description

La professió de Data Analyst destaca per la seva adaptabilitat, ja que cada conjunt de dades presenta desafiaments únics que requereixen creativitat i pensament analític. Aquest perfil professional se centra en proporcionar respostes ràpides i pràctiques per a la presa de decisions empresarials.

La profesión de Data Analyst destaca por su adaptabilidad, ya que cada conjunto de datos presenta desafíos únicos que requieren creatividad y pensamiento analítico. Este perfil profesional se centra en proporcionar respuestas rápidas y prácticas para la toma de decisiones empresariales.

The Data Analyst profession stands out for its adaptability, as each dataset presents unique challenges requiring creativity and analytical thinking. This professional profile focuses on providing quick and practical answers for business decision-making.

## Objectius del curs / Objetivos del curso / Course Objectives



**Habilitats / Habilidades / Skills:**

- Per depurar, analitzar, interpretar i calcular dades per extreure coneixements del negoci i facilitar la presa de decisions.

- Para depurar, analizar, interpretar y calcular datos para extraer conocimientos del negocio y facilitar la toma de decisiones.

- To clean, analyze, interpret, and calculate data to extract business insights and facilitate decision-making.  


- Coneixements d’eines i tècniques analítiques, i la capacitat d’utilitzar llenguatges com SQL i Python.

- Conocimientos de herramientas y técnicas analíticas, y la capacidad de utilizar lenguajes como SQL y Python.

- Knowledge of analytical tools and techniques, with the ability to use languages such as SQL and Python.
  

**Aprenentatges / Aprendizajes / Learnings**

- Treballar amb dades estructurades i comprendre models de dades no estructurades.

- Trabajar con datos estructurados y comprender modelos de datos no estructurados.

- Work with structured data and understand unstructured data models.

- Dur a terme anàlisis descriptives que generin coneixements útils per a la presa de decisions.

- Realizar análisis descriptivos que generen conocimientos útiles para la toma de decisiones.

- Perform descriptive analyses that generate valuable insights for decision-making.

- Exercicis pràctics amb datasets reals per a aplicació immediata.

- Ejercicios prácticos con datasets reales para aplicación inmediata.

- Practical exercises with real datasets for immediate application.
